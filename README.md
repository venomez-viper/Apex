# APEX-II / STAM

This repository contains the APEX-II research paper and experiment artifacts by
Akash Anipakalu Giridhar.

APEX-II extends the AI Presales Exposure Index with STAM: a deal-size-stratified,
subtask-level, TAM-aware simulation framework for studying AI exposure in SaaS
presales engineering.

# Reports

This directory contains human-readable reports supporting the APEX-II / STAM
paper. The reports are written to make the repository reviewable without forcing
readers to inspect raw pipeline logs first.

| Report | Purpose |
| --- | --- |
| `methodology_report.md` | Model design, variables, assumptions, and ablations |
| `experiment_report.md` | Execution summary and artifact map for the completed run |
| `results_report.md` | Main numerical findings and interpretation |
| `reproducibility_report.md` | How to rerun or audit the experiment |
| `quality_audit_report.md` | Stage 20 quality gate and artifact-grounding audit |
| `limitations_report.md` | Internal, external, and construct-validity limits |
| `conclusion_validation_report.md` | Maps paper conclusions to supporting artifacts |
| `ieee_preparation_checklist.md` | IEEE-style paper preparation checks |

Raw pipeline artifacts are preserved under `reports/pipeline/`.
